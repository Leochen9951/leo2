# 双人成行通关教程静态网页

这是一个新的静态网页项目，用于展示《双人成行》的中文通关路线、原创通关图解、章节攻略、常见卡点和一周目补漏清单。

## 与旧版项目的关系

旧版介绍页保留在：

`F:\Codexworkspace\projects\it-takes-two-site`

本项目为新建项目，不覆盖旧版文件：

`F:\Codexworkspace\projects\it-takes-two-walkthrough-site`

## 文件

- `index.html`：页面结构与攻略内容
- `styles.css`：响应式布局和视觉样式
- `script.js`：移动导航、章节筛选、搜索、进度勾选和滚动状态
- `assets/hero-coop-adventure.png`：复用旧项目中的原创生成主视觉
- `assets/guide-*.svg`：七大章节原创通关图解，不直接搬运外部截图

## 本地查看

直接用浏览器打开 `index.html` 即可。
